## General

- season : 시즌정보
- queueId : 게임모드 종류 (랭크, URF, 사설게임 ...등)
- mapId : 맵 종류
- gameMode : 게임모드 종류
- gameType : 게임 종류 (튜토리얼, 커스텀매치, 그 외)
